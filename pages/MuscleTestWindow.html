Muscle Test Window




This testing window is used to test the muscles which have been selected. Data collected from device used will produce a force curve after test trial is completed. Time length of 8 seconds is allowed to complete an exertion.


Buttons:


Test: Opens the Force Window. When a valid device is selected this will take you through an automated testing process. If the muscle is bilateral one side will be tested then the other. To jump to the opposite side before five exertions have been done, press the space bar while the Force Dialog box is up. Or, click Cancel in the Force Dialog box, then move the cursor to first trial on opposite side, and click Test to start testing.

Zoom: Opens the Zoom Curve Window.


Delete Trial: Deletes the selected exertion. All tests for that side that are located after the selection will be moved up.


Reset Trial: Deletes all completed exertions.


Export: Export button allows to save test information as a .csv file in current muscle test screen of completed muscle test trial(s). A Save As window displays. Type in file name how wish to save, and click OK button. You can open .csv file in spreadsheet program such as excel, to view saved test information, and force sample points for each trial conducted.


Single/Split Graph: Toggles the graph between single and split views.


Next: Next button brings up next selected muscle test.


Previous: Previous button brings up previous muscle test


Help: Brings up this help screen.


Quit: Recogize whether this is for readonly or for performing test. If for performing test then opens Validity or If this is for readonly then Closes the Muscle Test dialog box.


Sound: Select sound option for testing.


Picture: Enlarges thumbnail picture to a full screen picture.


Additional Info: Displays picture with muscle test information.



Keys

F2: Same as Test Button.


Page Down/Up: Moves to the Next/Previous selected muscle. Page number is shown in upper right corner or dialog box.


F6: Enlarges thumbnail picture to a full screen picture.
