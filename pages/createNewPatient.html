NEW/EXISTING PATIENT






Click NEW button in Home Screen to create new patient. Modules screen displays Pre Test Data window to enter new patient information. For existing patient selected, Pre Test Data information can be updated and saved for new test session.





Personal Tab

New Patient: Patient information is entered and Evaluator entered/selected. Patient information is entered when creating new patient for testing. Complete and save Cover Letter and Copy Reports To Tabs, then click OK to save Personal Tab information. Each Pre-Test Data tab information is saved separately.


Existing Patient : For new/subsequent tests with existing patient, click on Modules in Menu. Select a patient, either by start typing name in Search, for name to display and select patient. Or click on drop down arrow to view patient list and select patient. The selected existing patient will have in Pre Test Data Window, patient information from previous test session displayed, and current date showing. The time window is blank, and time can be entered. If other patient information has also changed or needs to be updated from previous test session for the Cover Letter or Copy Reports To tabs, enter and save updated information for the other tabs also. Updated existing patient information saves to, correlates with new test session.


Cover Letter Tab

Information entered in this tab is for contact to send reports to. Enter contact information and click the Save button. Contact information for Cover Letter tab is now saved. The Recall button opens to saved contacts entered to select from. The selected contact information is automatically entered in the cells in the tab. Click the Save button to save recalled contact for the Cover Letter Tab.


Copy Reports To Tab

Information entered in this tab for contact(s) to copy reports to. Enter contact information and click the Save button. Contact information is now saved for Copy Reports To Tab. The Recall button opens to saved contacts entered to select from. The selected contact information is automatically entered in the cells in the tab. Click the Save button to save recalled contact for Copy Reports To Tab.
