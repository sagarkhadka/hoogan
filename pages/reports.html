Reports




Letter/Narrative Tab


Generate Graphical Report: This button can be used to generate all graphical reports together except comparison report for test module session completed. Report pages will be created and screen will open to preview/view reports From print preview screen all report pages can be printed to the printer or saved using an installed PDF application to generate a PDF file.


Generate Report Button: This button is for generating Cover Letter or Narrative pages when one or both are selected. The pages will open in a word editor to review, edit. User determines how to name, save reports.


Print Preview Button: This button is used when comparison report is selected. Generated report will open to preview/view report. From print preview, comparison report can be viewed, and printed to the printer or saved using an installed PDF application to generate a PDF file.


Print Button: Print button goes right to print without print preview of reports, with selection of either print to the printer or save using an installed PDF application to generate a PDF file, when select Cover Letter, Narrative Letter, or Comparison report(s) in the tab.


Close Button: Closes the Reports window.


Save Defaults Button and Restore Defaults Button: The Save Defaults button is a time saving feature if you tend to print the same reports for all of your patients. Select those reports which you wish to print then select the Save Defaults. Then any time you want to print these reports on a patient just select the Restore Defaults button.


Help Button: Help button provides help information about Reports window.



Module Tabs


Modules Tabs: From Modules test tabs, graphic report options can be selected and previewed. The reports selected and generated can print to the printer or save using an installed PDF application to generate a PDF file.
