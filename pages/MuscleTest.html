Muscle Test


Before selecting Muscle Test in Modules for new test session, either create new patient and save information, or open existing patient, update and save information. After new or existing patient information is saved, click again on Modules in menu, and select newly created patient or updated existing patient from patient list for new test session. Next, click Muscle Test.


After click on Muscle Test button, check device connection in Device Connection window by clicking on device image displayed. After confirming connection, click Next button. Muscle Test Commands window displays.




Commands




Test: Use this button to create new muscle tests or view muscle tests that have already been performed. This button will open the Muscle Selection Window.




Protocols: Use this button to create new protocol and to edit or test previously created protocols. This button will open the Protocol Selection Window.




Recommendations: Use this button to select a recommendation for the patient. This button will open the Recommendations Window.




Repeat Test: Use this button for patients who are returning for further testing. For example, Mr. Jones has six muscles tested on January 15th. On January 22nd he returns to have the same six muscles test again. Instead of selecting the muscles all over again choose this button and you'll be taken straight to the testing screen with all six muscles already selected.




Exit: Use this button close the Muscle Test application.




Help: This button opens the main help screen.
