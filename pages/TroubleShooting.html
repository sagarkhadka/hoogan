Trouble Shooting

No communication between software and device:

  1. Go to Device Manager on computer, by right click on the “Start” button on the task bar, and select “Device Manager”. In Device Manager, if the “USB serial port” device in the ports section is not displayed, check the “other devices” for a “FTDI USB UART”. If it is located under the other devices section, then the driver is not installed.
    To update driver, right click on the FT232R USB UART, and select “update driver.” Then select “browse this computer” and then point to the downloaded USB driver folder provided by Hoggan Scientific as part of your software purchase. Point to the folder, and click ok. The driver should then install. This will have to be completed twice, once for the USB driver and once for the com port driver. You can also go to the FTDI Chip website at https://ftdichip.com/ , and download the USB driver from the website to install.

  2. Check to make sure wireless RF module for device is turned on. For Microfet2, if the decimal point in front of H/L on the right lcd display is not visible, then check to make sure the RF module is turned on. See user manual for instructions. For other Microfet devices, refer to user manual also.

  3. If the decimal point in front of H/L on the right display is not blinking on Microfet2, then check the number on the FET Stick dongle. It should match the serial number of the Microfet2 that you are connecting to. When everything is working correctly, the decimal point in front of the H on the Microfet2 LCD display should be on and blinking, and the LED on the FET Stick should be solid (not blinking). To check decimal point on other Microfet devices, refer to user manual.

  4. Any time the FET Stick is removed from the USB port while the software is open, wireless connection will be lost. Go back Modules screen after the FET stick is reinserted. Then reselect desired testing Module. When the Device Connection screen displays, click on device image in window to check communication connection. There may be some instances where program will need to be closed and reopened to regain connection.

  5. If have more than one Microfet device using with software, when preparing for testing, be sure that the serial number on the FET stick matches the serial number of device testing with. FET sticks are not interchangeable.

  6. Device does not power on: Rechargeable battery may need to be charged. Refer to Microfet device user manual for instructions on recharging battery. To avoid testing interruptions due to low battery power, we recommend that you check remaining battery power regularly, and re-charge battery when reaches approximately 15% power level. To check battery power, follow the battery check instructions in product user manual.
